# Blogify

A blogging platform for minimalists and developers.

https://blogify.dev

## Development

Generate a Prisma client:

```bash
yarn db-generate
```

Run database migration first:

```bash
yarn migrate-deploy
```

Start Server:

```bash
yarn dev
```

## Deployment

```bash
yarn db-generate
yarn migrate-deploy
yarn start
```
